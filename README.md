# Novel Time-Aware Food Recommendation System

A hybrid food recommendation system built using Django, deep learning, and graph clustering techniques to provide personalized and time-aware food recommendations based on user preferences, reviews, ratings, and interaction patterns.

## Features
- Hybrid recommendation engine combining collaborative and content-based filtering
- Time-aware recommendation modeling
- Graph clustering for user-food relationship analysis
- User authentication and profile management
- Food recommendation prediction module
- Recommendation analytics dashboard
- Interactive data visualization charts
- Dataset training and testing interface
- Recommendation ratio analysis
- Downloadable prediction datasets

## Tech Stack
- Python
- Django
- TensorFlow
- Pandas
- NumPy
- SQLite
- HTML/CSS
- CanvasJS

## Installation

### 1. Clone the Repository
```bash
git clone <your-github-repository-link>
```

### 2. Navigate to Project Directory
```bash
cd food-recommender-system
```

### 3. Install Required Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the Django Server
```bash
python manage.py runserver
```

### 5. Open in Browser
```txt
http://127.0.0.1:8000/
```
